# Personal Portfolio Website This is a personal portfolio website built with HTML, CSS, and JavaScript. The website features a modern design with a blue and green color theme, responsive layout, and interactive elements. ## Project Structure ``` ├── index.html # Main HTML file ├── assets/ │ ├── css/ │ │ └── style.css # Main stylesheet │ ├── js/ │ │ └── main.js # Main JavaScript file │ ├── vendor/ # Third-party libraries │ └── img/ │ ├── projects/ # Project images │ └── ventures/ # Venture images └── README.md # This file ``` ## Required Images ### Project Images Place your project images in the `assets/img/projects/` directory: - geothermal.jpg - coral.jpg - waste.jpg ### Venture Images Place your venture images in the `assets/img/ventures/` directory: - intrusivethoughts.jpg - cloudkitchen.jpg Recommended image sizes: - Project images: 800x600px - Venture images: 800x600px ## Dependencies The website uses the following third-party libraries: - Bootstrap CSS - jQuery - Remixicon - Boxicons - Typed.js - AOS (Animate On Scroll) ## Setup Instructions 1. Clone this repository 2. Add your images to the respective directories 3. Update the content in `index.html` with your information 4. Deploy to your preferred hosting service ## Customization ### Colors The color scheme can be modified in the CSS variables at the top of `style.css`: ```css :root { --primary-color: #2E8B57; /* Sea Green */ --secondary-color: #4682B4; /* Steel Blue */ --background-light: #F0F8FF; /* Alice Blue */ --text-dark: #2C3E50; /* Dark Blue */ --accent-green: #3CB371; /* Medium Sea Green */ --accent-blue: #5F9EA0; /* Cadet Blue */ } ``` ### Content Update the content in `index.html` to reflect your personal information, experience, projects, and ventures. ## Browser Support The website is compatible with: - Chrome (latest) - Firefox (latest) - Safari (latest) - Edge (latest) ## License This project is licensed under the MIT License - see the LICENSE file for details. :star: Star me on GitHub — it helps!
